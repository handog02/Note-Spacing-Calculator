    Instructions

    The first number you should enter is the amount of space you have to work with. This can be in any unit. Afterwards, notes can be entered from left to right. Use the reference below on how to type different note lengths. Everything should be separated by a forward slash (/). To add a space, for example, to allow for a bar line or a cautionary clef, type 'a' (for 'add') followed by the amount of space required in the same units as the number at the beginning, e.g. a5 to add five units of space. For large groups of the same type of note, it is possible to group them together by adding a 'x' followed by the number of notes onto the end, e.g. q.x4 will give four dotted quarter notes.

    Note reference

    • w: whole note
    • h: half note
    • q: quarter note
    • e: 8th note
    • s: 16th note
    • t: 32nd note

    For a dotted note, add a '.' to the end. Currently, these are the only notes supported.
